## Image Best Practices

### Alt Text Guidelines

- **Descriptive**: Provide meaningful descriptions for screen readers
- **Concise**: Keep alt text under 125 characters when possible
- **Contextual**: Describe the image's purpose in the content
- **Decorative**: Use empty alt="" for purely decorative images

### Image Optimization

- Use appropriate formats (JPEG for photos, PNG for graphics with transparency)
- Optimize file sizes for web performance
- Consider responsive images for different screen sizes
- Use meaningful filenames that describe the content

### Accessibility Considerations

- Ensure sufficient color contrast in images with text
- Avoid using images as the only way to convey important information
- Test with screen readers to ensure proper alt text
- Consider providing captions for complex images
